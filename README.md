# Object Architecture Exercise;

## 0 - Introduction

This is a program focused on managing sales, clients and products.

With it you can do all management operation such as adding, visualizing and modifying as well as to get generic information such as most product sold, client with most expenses, avarage price value of a product and so more.

## 1 - Dependecies

Program writen with pure java 25 SDK, no dependencies others than java's standard needed.

## 2 - How to run the project

To run the project execute it inside a IDE using the *src/java/main/Main* class, or execute the following command from your local project folder.

On Windows:

command: *java -classpath \PROJECT_FOLDER\target\classes main.Main*
